# Claude Skills

## Installation

- Clone this repo into your Claude Code skills directory:
  ```bash
  cd ~/.config/claude-code/skills/
  git clone <repository-url> .
  ```

- Or manually copy the skill folders to:
  - macOS/Linux: `~/.config/claude-code/skills/`
  - Windows: `%APPDATA%\claude-code\skills\`

- Restart Claude Code
- Check /skills to see the list of available skills

- Use skills with `/skill-name` or by asking Claude naturally

## Skills

- **write-tickets**: Write JIRA tickets for a project
